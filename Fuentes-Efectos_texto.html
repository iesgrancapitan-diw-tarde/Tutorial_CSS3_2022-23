TUTORIAL CSS3 - Fuentes y Efectos de Texto

Propiedades para modificar las fuentes y el texto de un documento HTML

color

Establece el color del contenido de los elementos seleccionados.

Sintaxis

color: [Aqui va la sintaxis de cada propiedad];

El color se puede escoger de distintas maneras:

  • rgb()
  • rgba()
  • hsl()
  • hsla()
  • Valores hexadecimales
  • Variables por defecto

Ejemplos:

p {
    color: rgb(255, 0, 0);
    color: rgba(255, 0, 0, 0.5);
    color: hsl(0, 100%, 50%);
    color: hsla(0, 100%, 50%, 0.5);
    color: #ff0000;
    color: red;
}

Las formas de rgba y hsla admiten el valor alpha, que indica la transparencia del color. El valor alpha puede ser un número entre 0 y 1, o un porcentaje entre 0% y 100%.

font-family

Esta propiedad indica al navegador el tipo de letra que debe aplicar a los elementos seleccionados. El navegador solo aplica el tipo de letra si la máquina que accede al sitio web dispone de ella; en caso contrario, simplemente usa el tipo de letra que tiene definido por defecto (default font).

Sintaxis

font-family: [Aqui va la sintaxis de cada propiedad];

Los tipos de letra seguros para la web:

  • Arial -> sans-serif
  • Courier New -> monospace
  • Georgia -> serif
  • Times New Roman -> serif
  • Trebuchet MS -> sans-serif
  • Verdana -> sans-serif

Ejemplos:

p {
    font-family: Arial;
}

CSS define cinco nombres genéricos para los tipos de letra serif, sans-serif, monospace, cursive y fantasy.

Nombre genérico Descripción
serif El tipo de letra tiene serifa (una pequeña línea que sobresale del borde de cada letra). Los tipos de letra serif son más tradicionales y se utilizan en documentos impresos.
sans-serif El tipo de letra no tiene serifa. Los tipos de letra sans-serif son más modernos y se utilizan en documentos web.
monospace El tipo de letra tiene todos los caracteres con el mismo ancho. Los tipos de letra monospace se utilizan para mostrar texto de código fuente.
cursive El tipo de letra se basa en una fuente de caligrafía. Los tipos de letra cursive se utilizan para mostrar texto de caligrafía.
fantasy El tipo de letra se basa en una fuente decorativa. Los tipos de letra fantasy se utilizan para mostrar texto decorativo.

font-size

Establece el tamaño de la fuente.

Sintaxis

font-siza: [Aqui va la sintaxis de cada propiedad];

El tamaño se puede escoger de distintas maneras:

  • Píxeles
  • Ccsme
  • Rems
  • Ex
  • %

Ejemplos:

p {
        font-size: 8px;
        font-size: 7em;
        font-size: 6rem;
        font-size: 5ex;
        font-size: 4%;
}

La unidad rem es igual al tamaño de fuente del elemento raíz en cambio em es igual al tamaño de fuente actual del elemento en el que se aplica.

font-style

Se usa para activar y desactivar el texto en cursiva.

Sintaxis

font-style: [Aqui va la sintaxis de cada propiedad];

Los valores posibles son los siguientes:

  • normal - Pone el texto en tipo de letra normal (desactiva la cursiva).
  • italic - Pone el texto en la versión cursiva del tipo de letra, si está disponible; si no está disponible, simulará cursiva con el modo oblique en su lugar.
  • oblique - Pone el texto en una versión de tipo de letra cursiva simulada, que se genera inclinando la versión normal.

Ejemplos:

p {
    font-style: normal;
    font-style: italic;
    font-style: oblique;
}
