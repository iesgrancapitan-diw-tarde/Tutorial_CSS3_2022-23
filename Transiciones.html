TUTORIAL CSS3 - Bordes

Propiedades para modificar los bordes de los elementos de un documento HTML

transition-duration

Indica el tiempo que debe tardar en completarse una animación de transición. Por defecto es 0, es decir, que no se producirá ninguna animación: transition-duration:0s;

Sintaxis

transition-duration: [tiempo de espera]s/ms;

Ejemplos:

En segundos:
.transition-duration-ejemplo-segundos {
    width: 100px;
    height: 100px;
    background: red;
    transition-duration: 4s;
}

.transition-duration-ejemplo-segundos:hover {
    height: 150px;
}

Resultado:


En milisegundos:
.transition-duration-ejemplo-segundos {
    width: 100px;
    height: 100px;
    background: red;
    transition-duration: 400ms;
}
              
.transition-duration-ejemplo-segundos:hover {
    height: 150px;
}

Resultado:

transition-delay

Indica el tiempo de espera (en segundos o milisegundos) antes de empezar una transición

Sintaxis

transition-delay: [tiempo de espera]s/ms;

Ejemplos:

En segundos:
.transition-delay-ejemplo {
    width: 150px;
    height: 150px;
    background: blue;
    transition-duration: 4s;
    transition-delay: 2s;
}

.transition-delay-ejemplo:hover {
    width: 300px;
}

Resultado:


En milisegundos:
.transition-delay-ejemplo {
    width: 150px;
    height: 150px;
    background: blue;
    transition-duration: 4s;
    transition-delay: 20ms;
}

.transition-delay-ejemplo:hover {
    width: 300px;
}

Resultado:

transition-property

Indica la propiedad o propiedades CSS a la que se le va a aplicar el efecto de transición. Debera especificar la duración de dicha animación, de lo contrario, por defecto sera 0 y no ocurrirá nada.

Sintaxis

transition-property: none|all|property|initial|inherit;

Esta propiedad puede tener los siguientes valores:

  • none: ninguna propiedad tendra un efecto de transisión
  • all: todas las propiedades tendran un efecto de transisión (es el valor por defecto)
  • property: define una lista separada por comas de nombres de propiedades CSS para el efecto de transición
  • initial: establece esta propiedad en su valor predeterminado.
  • inherit: hereda esta propiedad de su elemento principal.

Ejemplos:

.transition-property1 {
    width: 90px;
    height: 90px;
    background: green;
    transition-property: none;
    transition-duration: 2s;
}

.transition-property1:hover {
    width: 200px;
    height: 200px;
    font-size: 40px;
    background-color: cyan;
    color: gray;
}

.transition-property2 {
    width: 90px;
    height: 90px;
    background: green;
    transition-property: all;
    transition-duration: 2s;
}

.transition-property2:hover {
    width: 200px;
    height: 200px;
    font-size: 40px;
    background-color: cyan;
    color: gray;
}

.transition-property3 {
    width: 90px;
    height: 90px;
    background: green;
    transition-property: width, height,background-color;
    transition-duration: 2s;
}

.transition-property3:hover {
    width: 200px;
    height: 200px;
    font-size: 40px;
    background-color: cyan;
    color: gray;
}

Resultado:

transition-property1

transition-property2

transition-property3

transition-timing-function

Indica la curva de velocidad del efecto de transición. Es decir, indica lo lento o rápido que va a ir de principio a fin una transición

Sintaxis

transition-timing-function: linear|ease|ease-in|ease-out|ease-in-out|steps|step-start|step-end|cubic-bezier

Esta propiedad puede tener los siguientes valores:

  • linear: se trata de un efecto de transisión con la misma velocidad de principio a fin
  • ease: se trata de un efecto de transisión con un comienzo lento, luego avanza rápido, y por último lento de nuevo
  • ease-in: se trata de un efecto de transisión con un comienzo lento
  • ease-out: se trata de un efecto de transisión con un final lento
  • ease-in-out: se trata de un efecto de transisión con un comienzo y un final lento
  • steps(int,start|end): se trata de una función (paso a paso) a la que le pasas dos parámetros. El primer parámetro especifica el numero de intervalos de la función(debe ser mayor a 0). El segundo parámetro (opcional) es siempre el valor "start" o "end", y especifíca el punto en el que se va a producir ese cambio de valores dado en el primer parámetro. Si no se le pasa nada, por defecto tomará el valor "end"
  • step-start: se trata de un efecto de transisión equivalente a steps(1,start)
  • step-end: se trata de un efecto de transisión equivalente a steps(1,end)
  • cubic-bezier(n,n,n,n): se trata de un efecto de transisión al cual le permite definir sus propios valores en una función cubic-bezier.(Los posibles valores van de 0 a 1)
He separado los diferentes ejemplos para que se pueda ver con mas claridad

Ejemplos 1:

#transition-timing-function-linear {
    transition-timing-function: linear;
}
                            
#transition-timing-function-ease {
    transition-timing-function: ease;
}
                            
#transition-timing-function-ease-in {
    transition-timing-function: ease-in;
}
            
#transition-timing-function-ease-out {
    transition-timing-function: ease-out;
}
            
#transition-timing-function-ease-in-out {
    transition-timing-function: ease-in-out;
}

Resultado:

linear
ease
ease-in
ease-out
ease-in-out

Ejemplos 2:

#transition-timing-function-cubic-bezier1 {
    transition-timing-function: cubic-bezier(1,0,0.5,0);
}

#transition-timing-function-cubic-bezier2 {
    transition-timing-function: cubic-bezier(0,0.3,1,0.8);
}

#transition-timing-function-cubic-bezier3 {
    transition-timing-function: cubic-bezier(0.5,1,0.4,0.7);
}

#transition-timing-function-cubic-bezier4 {
    transition-timing-function: cubic-bezier(0.3,0.6,0.4,0.5);
}

#transition-timing-function-cubic-bezier5 {
    transition-timing-function: cubic-bezier(1,0.3,0.4,1);
}

transition

Es la forma abreviada de todo lo explicado anteriormente. "shorthand".

Sintaxis

transition: property duration timing-function delay

Nos permite usar todas las propiedades de transición de una forma mas rápida.

Ejemplos:


.transition-transition {
    width: 90px;
    height: 90px;
    background-color: blue;
    transition: all 5s linear 1s;

}

.transition-transition:hover {
    width: 200px;
    background-color: rosybrown;

}
