TUTORIAL CSS3 - Fuentes y Efectos de Texto

Propiedades para modificar las fuentes y el texto de un documento HTML

color

Establece el color del contenido de los elementos seleccionados.

Sintaxis

color: [Aqui va la sintaxis de cada propiedad];

El color se puede escoger de distintas maneras:

  • rgb()
  • rgba()
  • hsl()
  • hsla()
  • Valores hexadecimales
  • Variables por defecto

Ejemplos:

p {
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5);
color: hsl(0, 100%, 50%);
color: hsla(0, 100%, 50%, 0.5);
color: #ff0000;
color: red;
}

Las formas de rgba y hsla admiten el valor alpha, que indica la transparencia del color. El valor alpha puede ser un número entre 0 y 1, o un porcentaje entre 0% y 100%.

font-size

Establece el tamaño de la fuente.

Sintaxis

font-siza: [Aqui va la sintaxis de cada propiedad];

El tamaño se puede escoger de distintas maneras:

  • Píxeles
  • Ccsme
  • Rems
  • Ex
  • %

Ejemplos:

p {
        font-size: 8px;
        font-size: 7em;
        font-size: 6rem;
        font-size: 5ex;
        font-size: 4%;
}

La unidad rem es igual al tamaño de fuente del elemento raíz en cambio em es igual al tamaño de fuente actual del elemento en el que se aplica.
