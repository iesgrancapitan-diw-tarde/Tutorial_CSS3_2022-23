TUTORIAL CSS3 - Pseudoclases

Subtitulo

:active

La pseudoclase :active representa a un elemento que el usurio está activando. Generalmente, la activación se efectúa cuando el usuario pincha sobre el elemento con el botón izquierdo del ratón. Aunque válido para cualquier elemento, su uso se destina normalmente a enlaces y botones.

Sintaxis

elemento:active {
        /* Definición de estilos */
    }

Los estilos definidos por :active pueden ser anulados por las posteriores definiciones de :link, :hover o :visited. Es una buena práctica colocar :active después de todas las definiciones de estilo de los enlaces.

Ejemplos:

button:active {
        color: red;
        background-color: black;
    
    a:active {
        background-color: yellow;
        text-decoration: none;
    }
