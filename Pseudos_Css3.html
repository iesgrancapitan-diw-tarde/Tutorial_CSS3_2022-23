TUTORIAL CSS3 - Pseudos

Pseudoclases y pseudoelementos

:active

La pseudoclase :active representa a un elemento que el usuario ha activado, generalmente, haciendo clic sobre el elemento con el botón izquierdo del ratón. Se suele utilizar en botones y enlaces.

Sintaxis

elemento:active {
    /* Definición de estilos */
}

Los estilos definidos por :active pueden ser anulados por las posteriores definiciones de :link, :hover o :visited. Es una buena práctica colocar :active después de todas las definiciones de estilo de los enlaces.

Ejemplo de uso

button:active {
    color: red;
    background-color: black;
}

a:active {
    background-color: yellow;
    text-decoration: none;
}
Al pinchar en el enlace, se pondrá sobre fondo amarillo y no tendrá subrayado

:hover

La pseudoclase :hover representa a un elemento sobre el que el usuario ha colocado el cursor (o del dedo en dispositivos táctiles).

Sintaxis

elemento:hover {
    /* Definición de estilos */
}

Ejemplo de uso

button:hover {
    background-color: red;
    color: white;
}

:valid

La pseudoclase :valid representa a un elemento que contiene un valor válido. Se suele utilizar para validar formularios.

Sintaxis

elemento:valid {
    /* Definición de estilos */
}

Ejemplo de uso

input:valid {
    background-color: green;
    color: white;
}

:first-child

La pseudoclase :first-child representa a un elemento que es el primer hijo de su elemento padre.

Sintaxis

elemento:first-child {
    /* Definición de estilos */
}

Ejemplo de uso

    color: red;
}

::after

El pseudo-elemento ::after representa un elemento que se inserta después del contenido de su elemento padre.

Sintaxis

elemento::after {
    /* Definición de estilos */
}

Ejemplo de uso

    content: " - Texto añadido";
    color: green;
}
