TUTORIAL CSS3 - Pseudos

Pseudoclases y pseudoelementos

:active

La pseudoclase :active representa a un elemento que el usuario ha activado, generalmente, haciendo clic sobre el elemento con el botón izquierdo del ratón. Se suele utilizar en botones y enlaces.

Sintaxis

elemento:active {
    /* Definición de estilos */
}

Los estilos definidos por :active pueden ser anulados por las posteriores definiciones de :link, :hover o :visited. Es una buena práctica colocar :active después de todas las definiciones de estilo de los enlaces.

Ejemplo de uso

button:active {
    color: red;
    background-color: black;
}

a:active {
    background-color: yellow;
    text-decoration: none;
}
Al pinchar en el enlace, se pondrá sobre fondo amarillo y no tendrá subrayado

:hover

La pseudoclase :hover representa a un elemento sobre el que el usuario ha colocado el cursor (o del dedo en dispositivos táctiles).

Sintaxis

elemento:hover {
    /* Definición de estilos */
}

Ejemplo de uso

button:hover {
    background-color: red;
    color: white;
}

:valid

La pseudoclase :valid representa a un elemento que contiene un valor válido. Se suele utilizar para validar formularios.

Sintaxis

elemento:valid {
    /* Definición de estilos */
}

Ejemplo de uso

input:valid {
    background-color: green;
    color: white;
}

Estos elementos input son de tipo email, por lo que solo serán válidos si el valor introducido es un email:

:invalid

La pseudoclase :invalid representa a un elemento que contiene un valor no válido. Se suele utilizar para validar formularios.

Sintaxis

elemento:invalid {
    /* Definición de estilos */
}

Ejemplo de uso

input:invalid {
    background-color: red;
    color: white;

Estos elementos input son de tipo email, por lo que solo serán válidos si el valor introducido es un email:

:first-child

La pseudoclase :first-child representa a un elemento que es el primer hijo de su elemento padre.

Sintaxis

elemento:first-child {
    /* Definición de estilos */
}

Ejemplo de uso

ul > li:first-child {
    color: red;
}

:last-child

La pseudoclase :last-child representa a un elemento que es el último hijo de su elemento padre.

Sintaxis

elemento:last-child {
    /* Definición de estilos */
}

Ejemplo de uso

ul > li:last-child {
    color: green;
}

El último elemento de la lista aparece en verde:

:nth-child()

La pseudoclase :nth-child() representa a un elemento que es el hijo n-ésimo de su elemento padre.

Sintaxis

elemento:nth-child(n) {
    /* Definición de estilos */
}

Ejemplo de uso

ul > li:nth-child(2) {
    color: blue;
}

El segundo elemento de la lista aparece en azul:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

::after

La pseudo-elemento ::after representa un elemento que se inserta después del contenido de su elemento padre.

Sintaxis

elemento::after {
    /* Definición de estilos */
}

Ejemplo de uso

p::after {
    content: " - Texto añadido";
    color: green;
}
